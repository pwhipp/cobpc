This site is under construction and is here for testing and development purposes only.

Any domain name/url we own can be used to reference this. I can run it on one of my existing servers. We may be able to use the server currently being used for the website depending upon access/contract.

This is a web service built using Django. Its good for customization and extension (much better than Wordpress) but requires a bit more work initially. Like Wordpress, there are many third party additions.

I've not applied much in the way of styling but it can be styled however we like - everything is under our control.

This can:

  • Be accessed as a link from the main site - which can go straight to login (unless persisted).
  • Be styled however we like
  • Replace the original site if we add blogging/news etc. Might take me an elapsed month or two - benefits over time would include things like:
    • daily notices on the front page
    • rolling upcoming comps (with booking/status)
    • results (for members only)
Disadvantages of this approach
It can be set up for routine addition of articles and information etc. like wordpress but changes to functionality may involve some python programming requiring knowledge of django html and javascript.
